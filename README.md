# Getting Started with Beer Beer

React project created by using the Open Brewery DB.

## Installation
1. Clone and fork this repository.
2. npm install.

## Google Maps
1. Find or get your API key for Google Maps.
2. Create a .env file inside the Beer Beer folder.
3. Copy `REACT_APP_GOOGLE_MAPS_API_KEY='your API key'`.
4. Make sure you do not have '' around your API key.

Run your server and enjoy!
